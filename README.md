# orange-storyboard

把短片创意、广告概念或故事简报扩写成九宫格分镜提示词，并基于生成的故事板输出图生视频提示词。

## 工作流程

**第一步：生成故事板提示词**

描述你的创意，和skill说“生成故事板提示词”，skill 输出：
- 项目卡（风格、主角、场景、情绪目标等）
- 九宫格故事板生成提示词（可直接用于任意图像生成工具）
- 可选微调建议

比如
```
女孩在门口蹲下来和一只猫打招呼，门口的风铃在响，老式动画日系风格
```

```
高端香水 TVC，16:9，欧洲清晨古堡花园风格
```

**第二步：生成图生视频提示词**

拿到故事板图片后，发给 skill 并说"生成图生视频提示词"，输出：
- 视频总设定（时长、风格、节奏、声音方向）
- 9个镜头的逐格提示词（约15秒）

```
（上传故事板图片）生成图生视频提示词
```

## 参考图支持

可上传角色、产品、环境或风格参考图。Skill 会声明各图的控制范围并在每格分镜中保持一致。

## 支持风格

日系老式动画、手绘动漫、奢品TVC、明亮3D动画、古典武侠、赛博朋克、水墨国风、都市爱情、科幻未来、欧式童话、纪录片等14种风格，其他风格同样可生成。
